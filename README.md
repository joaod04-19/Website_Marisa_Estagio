# Website_Marisa_Estagio
Desenvolvimento de Website para empresa de Crianças.

# Descrição
O website apresenta as diferentes áreas de intervenção do Pigoitas, a equipa técnica, 
informações institucionais e um formulário de contacto com integração direta para o WhatsApp.
O design foi desenvolvido de forma responsiva para funcionar em computadores, tablets e dispositivos móveis. 
Sendo elaborado de forma simples e agradável aos olhas das crianças e dos encargados de educação.


# Funcionalidades
- Página Inicial com Apresentação do centro;
- Apresentação das Áreas de Intervenção;
- Apresentação da Equipa Multidisciplinar;
- Página Institucional "Sobre Nós";
- Marcação de Avaliação através de Formulário Integrado com WhatsApp;
- Galeria de Imagens;
- Design Responsivo para diferentes tamanhos de Ecrã;
- Elementos Gráficos decorativos em SVG;
- Navegação Intuitiva e Acessível;


# Tecnologias Utilizadas
- HTML;

- CSS;

- JavaScript;


# Bibliotecas Utilizadas e Recursos Externos
- Font Awesome (Ícones);

- Google Fonts (Nunito);

- SVGs personalizados para Elementos Decorativos (https://www.svgbackgrounds.com/elements/);


# Estrutura do Website
O website é composto pelas seguintes páginas:

### Home (index.html) – Página Principal do Website;
    - Hero Principal;
    - Áreas de Intervenção;
    - Centro de Estudos e Desenvolvimento Infantil;
    - Apresentação da Equipa;
    - Galeria de Fotos;
    - Contactos;

### Equipa (equipa.html) - Página da Equipa Completa;
    - Fotografia do Funcionário;
    - Função de cada um;
    - Biografia;
    - Áreas de Especialização;

### Sobre Nós (sobrenos.html) - Página para Apresentar a Empresa;
    - "Quem Somos?";
    - Valores e Principios;

### Avaliação (avaliacao.html) – Formuláio para Contactar com a Empresa;
    - Formulário de Contacto;
    - Introduzir dados da Criança e do Encargado de Educação;
    - Selecionar o Serviço Pretendido;
    - Escrever uma Mensagem Personalizada;
    - Integração com WhatsApp;
